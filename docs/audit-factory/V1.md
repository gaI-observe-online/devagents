## Purpose
Generate an auditable review pack for every PR, including:
- Findings summary
- Evidence and logs
- File inventory and diffs
- Recommendations and risks

## Gate
PR merges are blocked unless:
- `Code Review Factory (Audit Pack) / review_pack` passes

## Artifact
The workflow uploads:
- `review-pack-pr-<PR>.zip` (or `review-pack/**`)

## v1 Scope
- Deterministic packaging
- Artifact upload
- Basic CI reliability and diagnostics

## Out of Scope (for v1)
- Auto PR comments
- Auto-generated executive summary
- Risk scoring
- Policy as code
