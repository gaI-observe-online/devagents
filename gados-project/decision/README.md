# Decisions (Escalations)

Human authority decisions live here (e.g., `ESCALATION-009.md`).

## When to escalate
Escalate when a decision is:
- high-risk (security/privacy/safety)
- irreversible
- economic/pricing-changing
- unclear due to missing evidence or conflicting evidence

## Start here
Use `/gados-project/templates/ESCALATION.template.md`.
