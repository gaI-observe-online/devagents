# UI Screenshots Checklist (Optional)

Purpose: provide a lightweight checklist of screenshots for audits, demos, and change reviews.

## Control plane pages
- Dashboard (`/`)
  - Shows epic/story counts and status distribution
- Artifacts browser (`/artifacts`)
  - Shows `gados-project/` tree
- Create (`/create`)
  - Epic/story/change plan creation forms
- Reports (`/reports`)
  - Run daily digest; list generated report artifacts
- Inbox (`/inbox`)
  - Send message; pending messages; ACK/NACK controls
- Validate (`/validate`)
  - Governance validator results

## Evidence examples (artifact screenshots)
- A generated daily digest report under `gados-project/log/reports/`
- A populated bus audit log `gados-project/log/bus/bus-events.jsonl`
